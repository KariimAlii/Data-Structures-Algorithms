
# Forward Linked Lists

## Time Complexity

| Function | Location     | Time Complexity |
|----------|--------------|-----------------|
| Lookup   | By Value     | O(n)            |
| Lookup   | By Index     | O(n)            |
| Insert   | At Beginning | O(1)            |
| Insert   | At End       | O(1)            |
| Insert   | At Middle    | O(n)            |
| Delete   | Beginning    | O(1)            |
| Delete   | From End     | O(n)            |
| Delete   | From Middle  | O(n)            |
|          |              |                 |
